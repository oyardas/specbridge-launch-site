# SpecBridge Launch Site

This repository is for SpecBridge AI launch readiness only.

It is separate from the core technical product repository.

## Scope

- Landing page
- Domain and brand preparation
- Website content
- Legal / KVKK draft placeholders
- Demo and pilot materials
- GTM assets

## Out of Scope

- No changes to private technical repository
- No product patching
- No commit or push to the core repo
- No production SaaS claims
- No customer document collection before legal/security readiness

## Current Product Status

- Local/demo-ready MVP approaching
- DB read disabled
- External OpenAI disabled
- Live evidence lookup disabled
- LLM provider mock/dry-run
- JSON/file output authoritative
