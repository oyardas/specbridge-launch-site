# Production Readiness Boundary

Status: Draft
Scope: External communication boundary

## Current product state

SpecBridge AI is approaching a local/docker demo-ready MVP. It is not yet a production SaaS service.

Known current technical boundary:

- Local/docker environment
- Mock/dry-run LLM provider
- DB read disabled
- External OpenAI disabled
- Live evidence lookup disabled
- JSON/file output authoritative
- Report package generation available
- HTML, DOCX, PDF, XLSX, JSON outputs available
- Executive, Technical, and Sales report profiles available

## Website claim boundary

Allowed launch claims:

- Tender intelligence and presales decision support platform
- Demo-ready controlled MVP
- Generates structured report packages in controlled environments
- Supports executive, technical, and sales-oriented report views
- Designed for RFP/tender analysis workflows
- Intended to support bid/no-bid, compliance, risk, and clarification workflows

Avoid launch claims:

- Production SaaS
- Fully automated legal/commercial decision engine
- Guaranteed tender compliance
- Live public document processing
- Real-time external evidence verification
- Customer data stored securely in production cloud
- AI provider-backed production analysis

## Public demo boundary

Allowed:

- Screenshots
- Static sample report
- Anonymous sample output
- Five-minute demo script
- Controlled live demo from local environment
- No customer document upload through website

Not allowed yet:

- Public upload button
- Public trial workspace
- Self-service account creation
- Payment page
- Customer file processing through public website
- Production API claims

## Pilot boundary

Pilot may be offered only as a controlled engagement.

Pilot terms should define:

1. Pilot duration
2. Number of documents
3. Data handling model
4. Confidentiality
5. Retention and deletion
6. Output limitations
7. Human review requirement
8. Success criteria
9. Commercial follow-up path
10. No guarantee language

## Decision

The launch website should present SpecBridge AI as a controlled demo and pilot-stage product, not as an open public SaaS.
