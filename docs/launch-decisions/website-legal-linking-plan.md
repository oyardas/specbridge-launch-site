## Public launch wording

Avoid saying:

- Fully KVKK compliant
- Legally approved
- Certified secure
- Production-ready SaaS
- Automated legal decision engine

Use:

- Draft policy under review
- Pilot-stage data handling model
- Controlled demo process
- No public file upload at launch
- Customer documents processed only under agreed pilot terms

## Website integration decision

For the first public launch:

1. Convert legal Markdown files into static website pages.
2. Add footer links on every public page.
3. Keep the contact model email-only.
4. Add a visible warning that confidential tender files should not be sent before a pilot agreement.
5. Do not add cookie banners unless analytics/cookies are actually enabled.
