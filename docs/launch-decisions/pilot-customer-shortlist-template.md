# Pilot Customer Shortlist Template

Status: Draft
Product: SpecBridge AI
Scope: Controlled pilot planning

## Pilot target profile

Ideal pilot customer:

- Has recurring tender/RFP workload
- Needs presales or compliance review support
- Can provide anonymized or low-sensitivity sample documents
- Accepts controlled pilot boundaries
- Has clear success criteria
- Can provide structured feedback
- Does not require production SaaS access at this stage

| Criterion | Weight | Notes |

## Pilot entry criteria

Do not start a pilot unless:

1. Pilot scope is written
2. Document sensitivity is known
3. Retention and deletion terms are accepted
4. Output limitations are accepted
5. Human review requirement is accepted
6. Success criteria are agreed
7. Commercial next step is defined
